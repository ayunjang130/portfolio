# Saitama Portfolio - WWW Bootcamp Project

This is a web portfolio project featuring Saitama from "One Punch Man" created as part of the WWW Bootcamp curriculum.

## Technologies Used

- **HTML5** - Markup language for structuring the content
- **CSS3** - Styling and layout with modern features like CSS variables, flexbox, and media queries
- **JavaScript (ES6+)** - Client-side scripting for interactive elements
- **Webpack** - Module bundler for building and optimizing the project
- **Babel** - JavaScript compiler for backward compatibility
- **Node.js** - JavaScript runtime environment for development tools

### Webpack Plugins and Loaders
- `html-webpack-plugin` - Automatically generates HTML files and injects bundled assets
- `css-loader` & `style-loader` - Process and inject CSS files
- `babel-loader` - Transpiles modern JavaScript using Babel
- `webpack-dev-server` - Development server with live reloading

## Project Structure

```
.
├── index.html          # Main HTML template (without script tags)
├── src/
│   └── index.js        # Main JavaScript file with all application logic
├── public/             # Static assets (images)
│   ├── saitama1.png
│   ├── saitama2.png
│   └── saitama3.png
├── dist/               # Build output directory (generated by webpack)
├── webpack.config.js   # Webpack configuration file
└── package.json        # Project dependencies and scripts
```

## Features

- Responsive design that works on mobile and desktop
- Image carousel for showcasing multiple Saitama images
- Modal contact dialog
- Modern CSS with variables and flexbox layout
- Webpack build process for optimization

## Development

### Prerequisites
- Node.js (v12 or higher)
- npm (usually comes with Node.js)

### Setup
```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install
```

### Available Scripts
```bash
# Build the project for production
npm run build

# Start development server with live reloading
npm run dev
```

After running `npm run build`, the production files will be available in the `dist/` directory.

## Deployment

This project can be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages. When deploying to Vercel, set the output directory to `dist`.

## About

This project was created as part of the WWW Bootcamp curriculum to demonstrate modern web development techniques including HTML, CSS, JavaScript, and build tools like webpack. The portfolio showcases Saitama, the protagonist of "One Punch Man", featuring his training routine and powers.
